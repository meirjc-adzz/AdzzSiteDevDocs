We use Tailwind CSS for the styling of the Adzz Site. The theme settings are defined in the `tailwind.config.js` file. The theme settings include the color palette, typography, and other settings.

<Aside>
To avoid future conflicts and to maintain consistency, please use the color palette and typography settings defined in the `tailwind.config.js` file. this will help to maintain the consistency of the Adzz Site.
</Aside>

## Typography

The typography settings are defined in the `tailwind.config.js` file. The typography settings include the font family, font size, line height, and other settings.
